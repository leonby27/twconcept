# Timeweb concept

Лендинг по мотивам макета Figma «Timeweb. Website» (file key `Z1HMlhKYOQ6gcnKut6b0wO`, десктопный фрейм 1432px). **Чистый статический сайт: HTML + CSS + ванильный JS, без сборки и фреймворков.** Раньше был на Next.js — упрощён до статики (вся разметка в одном `index.html`, вся стилизация в `styles.css`, интерактив в `app.js`).

## Запуск

Любой статический сервер из корня, напр.: `python3 -m http.server 8090` → http://localhost:8090. Сборки нет.

## Структура

- `index.html` — вся разметка страницы (header + main со всеми секциями). Шрифты Inter/Roboto подключены через Google Fonts в `<head>`.
- `styles.css` — вся стилизация (BEM, токены в `:root`). Это бывший `globals.css`.
- `app.js` — вся интерактивность на ванильном JS (один IIFE): бургер-меню, аккордеон FAQ, переключатель периода в тарифах, выбор сценария в финальном CTA (рекомендации тарифов), «фонарик» в карточке «Почему», параллакс декора hero, появление блоков по скроллу.
- `assets/` — все ассеты (картинки, SVG-иконки Phosphor в `assets/icons/{duotone|solid}/`, лого брендов, фото, иллюстрации).
- `docs/PLAYBOOK.md` — принципы дизайна/структуры будущих блоков; `docs/MEDIA.md` — реестр фото/иллюстраций.

## Деплой

GitHub Pages, ветка `gh-pages` (classic Pages, корень). Сайт: https://leonby27.github.io/twconcept/. Пути к ассетам **относительные** (`assets/...`) — поэтому работают и на подпути `/twconcept/`. Деплой: скопировать `index.html`, `styles.css`, `app.js`, `assets/` (+ `.nojekyll`) в корень ветки `gh-pages` и запушить.

## Как добавлять/править секции

0. Контент от владельца (скрины — берём **только контент**, стиль не копируем; emoji не переносим — вместо них иконки Phosphor). **Перед вёрсткой — дизайн-проход из PLAYBOOK.** Акцент бережём (см. ниже).
1. Разметка — в `index.html` (серверного рендера нет, всё статично). Интерактив — в `app.js`. Стили — блоком в `styles.css` с комментарием-заголовком.
2. Иконки Phosphor: `<span class="icon" style="--icon: url(assets/icons/duotone/<name>.svg)">`. Докачка: `curl -sfL -o assets/icons/duotone/<name>.svg "https://unpkg.com/@phosphor-icons/core@2.1.1/assets/duotone/<name>-duotone.svg"`.
3. Фото — `<img loading="lazy">`, слот регистрируем в `docs/MEDIA.md`.

## CSS-конвенции

- BEM: `.hero__stat-value`, модификаторы `--` (`.btn--primary`).
- Цвета, шрифты, радиусы, гаттеры — только токены из `:root`. Новые значения — сначала токеном.
- Контентная ширина: `.container` или `width: min(var(--container-max), 100% - var(--page-gutter) * 2)`; `--container-max: 1214px`.
- Шрифты: заголовки/цифры — `--font-heading` (Inter), текст/UI — `--font-text` (Roboto). Подключены через Google Fonts; `--font-inter`/`--font-roboto` заданы в `:root`.
- Фоны секций чередуем: белый / `--color-surface`.
- **Акцент `#DC6655` бережём**: только CTA, сигнал «рекомендуем» и текст-ссылки. Функциональные инлайн-иконки — `--color-icon` (тёплый ink), НЕ акцент. Брендовые duotone-плитки (`.icon-tile`) остаются акцентными. Зелёный `--color-success` — только деньги/подарки/успех.
- Карточки на белом — с `box-shadow: var(--shadow-card)`. Единый сигнал выделения — акцентная рамка + `var(--shadow-accent)`.

## Адаптив (desktop-first)

Брейкпоинты `max-width`: **1280 (lg)** компактные гаттеры · **1024 (md)** бургер-меню, переносы рядов · **768 (sm)** типографика вниз, сетки 2×2 · **480 (xs)** колонки, full-width CTA, гаттеры 16px. Медиазапросы — в конце блока стилей секции.

Типографика по брейкпоинтам: H1 56/64 → 40/48 (sm) → 32/38 (xs); Display 40/48 → 32/40 → 28/34; подзаголовок 20/24 → 18/24 → 16/22.

## Анимации (правила)

Только лёгкие эффекты, всё в `app.js` + CSS. Reveal-по-скроллу (класс `.reveal` → `.is-revealed`), «фонарик» в `.why__hero` (маска `::before` по `--mx/--my`), параллакс декора hero, hover-микроанимации, аккордеон FAQ, фейд карт в рекомендациях. Только transform/opacity; `prefers-reduced-motion` уважается (JS-эффекты выключаются, CSS-микроанимации обёрнуты в media-запрос).
