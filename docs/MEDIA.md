# Медиа-реестр

Фото и иллюстрации генерирует/отдаёт владелец проекта отдельно. Здесь — что нужно, в каком виде и для какого блока. При вёрстке блока ставим заглушку и добавляем/уточняем строку здесь.

## Конвенция заглушек

- В разметке: `<div class="media-placeholder" data-media="<id>">` с подписью что здесь будет.
- Стиль заглушки: фон `--color-surface`, рамка дашем, по центру — название слота. (Класс добавим при первом использовании.)
- Файлы кладём в `public/assets/photos/` (фото) и `public/assets/illustrations/`, имя = id слота.
- Форматы: фото — WebP/AVIF + fallback JPEG, 2x под ретину; иллюстрации — SVG, если возможно.
- Рендер через `next/image` (фото) с шириной/высотой из таблицы.
